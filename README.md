### Instructions

#### To run some examples using [Gradle](https://gradle.org/)

```bash
gradle test
```

#### To run some examples using [Java](https://www.java.com/en/)

```bash
javac ${java-file}
java -ea ${java-file}
```

Where:

* `${java-file}` is the Java file you want to execute
